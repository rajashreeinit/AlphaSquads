  • Passwords: PBKDF2-SHA256 with salt
  • Tokens: JWT HS256 signed
  • Claims: UserId, Email, Name, Role
  • Expiry: 60 minutes (configurable)
  • Validation: Email uniqueness checked

⚠️  IMPORTANT - PRODUCTION CHANGES
  1. Change JWT SecretKey in appsettings.json (min 32 chars)
  2. Set ASPNETCORE_ENVIRONMENT=Production
  3. Use HTTPS only
  4. Use real database (SQL Server, PostgreSQL) instead of SQLite
  5. Enable CORS for specific origins only
  6. Add rate limiting
  7. Add logging to file

🐛 TROUBLESHOOTING

  Q: "dotnet: command not found"
     A: Install .NET SDK from https://dotnet.microsoft.com/download

  Q: "Port 5000/5001 already in use"
     A: Change port in Properties/launchSettings.json or kill process

  Q: "Database locked"
     A: Close other connections or delete auth.db and restart

  Q: "JWT validation failed"
     A: Check SecretKey in appsettings.json matches
